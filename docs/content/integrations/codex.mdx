## Open with AI

From the editor, **Open with AI ▸ Codex** dispatches the current file, folder, or project to Codex in one click. Two entry points: the sidebar's right-click menu (file / folder / empty space) and the macOS **File → Open with AI** menu. You can also hand the current doc to Codex from the bottom **Ask AI** composer.

Codex receives a short prompt asking it to open the target in Open Knowledge's web preview. The agent then grounds the rest of the conversation through MCP. See [Editor → Open with AI](/docs/features/editor#open-with-ai) for the full UX.
