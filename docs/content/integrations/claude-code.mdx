Open Knowledge works with Claude Code Agent via MCP and Skills.

## Install

<McpInstall editor="Claude Code" />

## Verify

Open the project in Claude Code and ask:

<VerifyExec subject="Claude Code" />

If Claude Code doesn't use the Open Knowledge `exec` tool, restart Claude Code so it picks up the new MCP entry.

## Live editing

When Claude Code writes to a doc, You will be able to see that change in the Open Knowledge app. Just ask Claude to open it for you.
